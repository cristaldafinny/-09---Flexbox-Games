Níveis

Flexbox Froggy

Nível 12: Desafio com `justify-content: space-around`.

Foi desafiador porque os espaços ao redor do conteúdo confundem quando se está começando. Exige atenção ao espaçamento entre os elementos e bordas.

Nível 18: Desafio com `align-items: flex-end` e `justify-content: center`.

Manipular os eixos simultaneamente exige entender bem o comportamento do eixo principal vs. eixo cruzado.

Nível 24: Uso de várias propriedades simultaneamente.

Foi o mais desafiador pois exigia o uso de `flex-direction`, `align-items`, `justify-content` juntos para acertar a posição dos sapos.

Flexbox Adventure

Fase 5: Uso de `flex-wrap: wrap` com múltiplos elementos.

Difícil por envolver quebra de linha e distribuição de espaço ao mesmo tempo.

Fase 7: Combinação de `justify-content: space-between` com `align-items: center`.

Desafiador pois o alinhamento dos cavaleiros nas duas direções exige bastante atenção visual.

Fase 10: Aplicação de `flex-direction: column`.

Foi um ponto de virada, pois a direção muda e afeta completamente o raciocínio espacial anterior.

Aplicação

Propriedades CSS utilizadas:

  • display: flex – Ativa o Flexbox no container
  • justify-content – Alinha os itens no eixo principal
  • align-items – Alinha os itens no eixo cruzado
  • flex-direction – Define a direção dos itens (linha ou coluna)
  • flex-wrap – Permite que os itens quebrem linha
  • align-self – Alinhamento individual de um item

Aplicação Prática:

Imagine um site com um menu de navegação responsivo. Com Flexbox, posso alinhar os itens na horizontal no desktop com `justify-content: space-between` e no mobile mudar com `flex-direction: column`, empilhando os links de forma responsiva e legível.
